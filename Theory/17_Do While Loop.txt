Do-While Loop -> A do-while loop is used to execute a block of code repeatedly until a condition becomes false.

It is called an exit-controlled loop because the condition is checked after the loop body executes.

->Syntax
    do {
        // Code to execute
    } while (condition);
->Example
    int i = 1;
    do {
        System.out.println(i);
        i++;
    } while (i <= 5);
Output
    1
    2
    3
    4
    5

->Important Point : The do-while loop executes at least once, even if the condition is false.

    int i = 10;
    do {
        System.out.println(i);
    } while (i < 5);
Output
    10

->Flow
    Execute Block
          ↓
     Check Condition
          ↓
       True → Repeat
          ↓
      False → Exit

->Difference Between while and do-while
    while	                        do-while
    Checks condition first          Executes first, then checks condition
    May execute 0 times             Executes at least 1 time
    Entry-controlled loop           Exit-controlled loop

->Interview Definition : A do-while loop is an exit-controlled loop that executes a block of code at least once and then continues execution as long as the specified condition is true.

->Quick Memory Trick
    while    → Check First, Execute Later
    do-while → Execute First, Check Later
