Logical Operators -> Logical operators are used to combine or manipulate boolean (true/false) expressions.

Operator	        Meaning	        Example
&&	                Logical AND	    a > 5 && b > 5
||                  Logical OR      a > 5 || b < 10
`		                            `
!	                Logical NOT	    !(a > 5)

->Truth Table
    A	        B	        A && B	    A || B
    true	    true	    true	    true
    true	    false	    false	    true
    false	    true	    false	    true
    false	    false	    false	    false

->Example
    int a = 10, b = 20;
    System.out.println(a > 5 && b > 5); // true
    System.out.println(a > 15 || b > 5); // true
    System.out.println(!(a > 5)); // false
    
->Interview Definition -> Logical operators are used to combine or reverse boolean expressions and return a boolean result (true or false).

->Quick Memory Trick
    &&  → Both conditions must be true
    ||  → At least one condition must be true
    !   → Reverses the result
