Data Types in Java -> A data type specifies the type of data that a variable can store.

->Example
    int age = 25;
    double salary = 50000.50;
    char grade = 'A';

->Types of Data Types
    Data Types
    │
    ├── Primitive
    │   ├── byte
    │   ├── short
    │   ├── int
    │   ├── long
    │   ├── float
    │   ├── double
    │   ├── char
    │   └── boolean
    │
    └── Non-Primitive (Reference)
        ├── String
        ├── Array
        ├── Class
        ├── Interface
        └── Object

1. Primitive Data Types

->Java has 8 primitive data types.

Data Type	        Size	        Example
byte	            1 byte	        byte b = 10;
short	            2 bytes	        short s = 100;
int	                4 bytes	        int age = 25;
long	            8 bytes	        long num = 100000L;
float	            4 bytes	        float f = 10.5f;
double	            8 bytes	        double d = 10.5;
char	            2 bytes	        char ch = 'A';
boolean	            JVM dependent	boolean flag = true;

->Example
    public class DataTypes {
        public static void main(String[] args) {
            byte b = 10;
            short s = 100;
            int i = 1000;
            long l = 10000L;
    
            float f = 10.5f;
            double d = 20.5;
    
            char ch = 'A';
            boolean flag = true;
    
            System.out.println(i);
        }
    }

2. Non-Primitive Data Types -> These store references (addresses) of objects.

->Examples
    String name = "Akash";
    int[] arr = {10, 20, 30};
    Object obj = new Object();

->Characteristics
    Created by programmers or Java libraries.
    Can call methods.
    Default value is null.

->Primitive vs Non-Primitive
        😳Primitive	            😳Non-Primitive
    Stores actual value	    Stores reference/address
    Predefined by Java	    Created by user/Java API
    Fixed size	            Size varies
    Cannot call methods	    Can call methods
    Faster	                Slightly slower

->Memory Trick
    Numeric Types
        byte → short → int → long
                        ↓
                   float → double
    Other Types
        char     → Single Character
        boolean  → true / false
