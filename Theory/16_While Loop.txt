While Loop -> A while loop is used to execute a block of code repeatedly as long as a condition is true.

It is generally used when the number of iterations is not known in advance.

->Syntax
    while (condition) {
        // Code to execute
    }
Example
    int i = 1;
    while (i <= 5) {
        System.out.println(i);
        i++;
    }
Output
    1
    2
    3
    4
    5
->Flow
    Condition
        ↓
      True
        ↓
     Execute Block
        ↓
     Update
        ↓
    Condition Again
        ↓
     False → Exit Loop

->Important Point : If the condition is false initially, the loop body will not execute even once.
    int i = 10;
    while (i < 5) {
        System.out.println(i);
    }
Output
    No Output

->Interview Definition : A while loop is an entry-controlled loop that repeatedly executes a block of code as long as the specified condition remains true.

->Quick Memory Trick
    while → Check Condition First
    True  → Execute Loop
    False → Exit Loop
